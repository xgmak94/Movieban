# MovieBan

Live at https://movieban.vercel.app/.

This is a kanban board like app to keep track of your favorite movies(or dislikes).

Utilizes the TMDB API to receive information on movies.

Implements email, Google, GitHub authentication.

Drag and drop functionality.

Implements light/dark mode switching.
